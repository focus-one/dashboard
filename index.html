일상글 관리
카페별 일상글 재고 현황 · 제품글 전환 관리
시트 열기
발행 시작 설정 ⓘ
어떤 카페
  • URL 시트(1zZkqN1_) 읽어 A열 분류=선택한 카페 + C열 사용가능여부=O인 첫 URL로 발행
어떤 제품
  • 계정 시트(16RS6FA / 2.일상글) B열 제품분류 매칭
  • 발행 로그 시트 C열 제품명에 그대로 기록
ID 개수 × ID당 일상글 개수
  • 총 N×M 글 발행. IP는 ID 바뀔 때만 변경.
  • 예: ID 2 × 글 3 = 6글 (계정A로 3글 연속 → IP 변경 → 계정B로 3글)
계정 선별 조건 (모두 AND)
  • A 상태 = O
  • B 제품분류 = 선택한 제품
  • C 사용할 카페 = 선택한 카페
  • D 포스팅 날짜 = 빈값 또는 24시간 이전 (24h 지나면 자동 재사용)
고급 설정
  • 동작 대기 / 계정 전환 대기 시간은 하단 토글에서 조정
사용 가능 계정: —
고급 설정 — 동작/계정 전환 대기 시간
